A powerful collection of custom Gutenberg blocks with advanced controls, reusable inspector panels, and a template library. Build stunning pages faster.

== Description ==

**GutenX Blocks** is a lightweight, performance-focused Gutenberg block library that gives you complete design control without leaving the WordPress editor.

= 7 Production-Ready Blocks =

1. **Advanced Heading** — Full typography controls with H1–H6 tag selector, font family, weight, spacing, and color options.
2. **Advanced Button** — Customizable button with size presets, background options, border controls, and hover effects.
3. **Container** — Versatile wrapper block with Flexbox/Grid layout, background images, and responsive controls.
4. **Row** — Multi-column row with gap controls, vertical/horizontal alignment, and mobile stacking.
5. **Icon Box** — SVG icon with heading and description, multiple layout options, and icon shape controls.
6. **Testimonial** — Customer quote card with avatar upload, star rating, and multiple layout styles.
7. **Image Gallery** — Responsive image gallery with Grid/Masonry layouts, lightbox, and caption support.

= Key Features =

* **Shared Inspector Controls** — Consistent Spacing, Typography, Background, and Border panels across all blocks.
* **Template Library** — 5 pre-designed layout templates with one-click import.
* **Admin Dashboard** — Modern React SPA to enable/disable individual blocks.
* **Performance First** — Each block loads only its own CSS. No global frontend styles.
* **Developer Friendly** — Clean OOP PHP, PSR-4 autoloading, REST API, and extensible hooks.

== Installation ==

1. Upload the `gutenx-blocks` folder to `/wp-content/plugins/`.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Navigate to **GutenX Blocks** in the admin sidebar to manage block settings.
4. Open the Block Editor and find all GutenX blocks under the **GutenX Blocks** category.

== Frequently Asked Questions ==

= What is the minimum WordPress version required? =
GutenX Blocks requires WordPress 6.3 or higher and PHP 8.0 or higher.

= Can I disable specific blocks? =
Yes. Go to **GutenX Blocks** in your admin menu and toggle any block on or off.

= Does this plugin work with any theme? =
Yes. GutenX Blocks is theme-agnostic and works with any block-based or classic theme.

= Will this slow down my site? =
No. Each block only loads its own CSS on pages where it is used. There are no global frontend scripts.

= Is there a Pro version? =
A Pro version with additional blocks, hover animations, carousel testimonials, and cloud template library is planned for a future release.
